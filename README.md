# 🧑‍💼 User Manager (React + TypeScript + Ant Design)

Цей проєкт — це простий SPA для керування користувачами з можливістю перемикання між режимами адміністратора та користувача.

## 🚀 Технології
- ⚛️ React (функціональні компоненти)
- 🧠 TypeScript
- 🎨 Ant Design
- 🔄 Context API
- 💾 LocalStorage

## 📦 Встановлення

```bash
